# my-dumb-html
This project is for the lulz only. Its simple and yeah dumb. Its a chemistry quiz and it uses the defauld old UI every html gives you if you dont tweak it.
To make your own version and a live site at [username].github.io/my-dumb-html/ follow these steps!

1. Make a fork of this repository
2. Go to index.html and click the file name or pencil
3. Make your changes
4. Save
5. Go to settings then Pages then under Build and deployment and leave deploy from a branch THEN under branch select main and make sure it is /(root and save!

Thats it! Have fun!
